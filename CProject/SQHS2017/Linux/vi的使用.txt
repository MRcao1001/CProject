1、vi 简介
		vi是visual interface(虚拟界面)的简称，它是UNIX世界中最通用的全屏幕
	文本编辑器。可以执行输出、删除、查找、替换、块操作等众多文本操作。而且
	用户可以对其进行定制，这是其他编辑器所没有的。
		vi没有菜单，它是通过命令来进行编辑操作的。
		Ubuntu Linux中，缺省的vi是vim，它兼容所有标准vi操作，并且有多窗口
	编辑，多风格显示等新功能。

2、vim的安装
	sudo apt-get install vim
3、vi的模式
	三种基本模式：命令模式、文本输入模式和末行模式
	(1)命令模式
			任何时候，不管用户处于何种模式，只要按一下ESC键，即可进入vi的命令
		模式。在shell环境下，输入vi命令，进入编辑器时，也是处于该模式。
			在该模式中，用户可以输入各种合法的vi命令，用于管理自己的文档。此时
		输入的任何字符，都被当做编辑命令来解释。若输入字符是合法的vi命令，则vi
		接收用户命令之后，完成相应的操作。
	(2)输入模式
			在命令模式中输入插入命令i、附加命令a、打开命令o、修改命令c、取代命令r
		或替换命令s，都可以进入文本输入模式。在该模式中，用户输入的任何字符都被
		vi当做文件的内容保存起来，并将其显示在屏幕上。在文本输入过程中，若想回到
		命令模式，按ESC键即可。
	(3)末行模式
			在命令模式中，用户按":"键即可进入末行模式。此时，vi会在显示窗口的最后
		一行显示一个":"作为末行模式的提示符，等待用户输入命令。多文件管理命令都是
		在此模式中执行的。末行命令执行完之后，vi将自动回到命令模式。
4、vi编辑器的基本命令
	(1)进入与离开
		进入vi可在系统命令提示符$之后输入: vi filename
			vi 可以自动载入想要编辑的文件，或是打开一个新文件。进入vi之后，在屏幕
		左方出现~，若行首有该符号，表明该行是空的。
			若没有输入文件名，则会打开一个空文件，并显示vi的简单帮助提示。

		离开:
			在命令模式中输入 	:q
			其实是先键入":"进入末行模式，然后使用q指令退出vi编辑器，如果文件被修改
		则应保存后退出，使用w命令保存文件，可以和q命令连用，表示保存后退出。即：:wq
	(2)命令模式的编辑操作
		移动光标：上下左右键，或是:k、j、h、l
		Ctrl+b：在文件中上移一页(相当于pageup)
		Ctrl+f：在文件中下移一页(相当于pagedown)
		H: 将光标移到屏幕的最上行
		nH：将光标移到屏幕的第n行
		M：将光标移到屏幕的中间
		L：将光标移到屏幕的最下行
		nL：将光标移到屏幕的倒数第n行
		W：在指定行内右移光标，到下一个字的开头
		E：在指定行内右移光标，到一个字的末尾
		B：在指定行内左移光标，到前一个字的开头
		0：数字零，左移到本行的开头
		$：右移到本行的行末
		^：移动光标，到本行的第一个非空字符
	(3)替换和删除
		将光标位于文件的指定位置之后，可以用其他字符来代替光标所指向的字符
		或从当前光标位置删除一个或多个字符。
		rc：用c替换光标所指向的位置
		nrc：用c替换光标所指向的当前字符开始的n个字符
		x：删除光标所指向的当前字符
		nx：删除光标所指向的前n个字符
		dw：删除光标右侧的字
		ndw：删除光标右侧的n个字
		db：删除光标左侧的字
		ndb：删除光标左侧的n个字
		dd：删除光标所在行
		ndd：删除n行内容，并去除空行
	(4)粘贴复制
		从正文中删除的内容并没有真正的丢失，而是被剪切并复制到内存缓冲区中。用户
		可以将其粘贴带正文中指定的位置。
		p：将缓冲区的内容粘贴至光标的后面
		P：将缓冲区的内容粘贴至光标的前面
		若缓冲区内容为整行正文，则粘贴在当前光标所在行的上一行或下一行。
		提示：
			vi编辑器中经常以一对大小写字母提供一对类似的功能。通常小写字母在光标
			的后面进行操作，大写命令在光标前面进行操作。

			yy：复制当前行的内容到缓冲区
			nyy：赋值n行内容到内存缓冲区
	(5)搜索字符串
		1)键入字符/
		2)后面跟要搜索的字符串
		3)然后按回车键
			编辑程序执行正向搜索，并在找到指定字符串之后，将光标停在该字符串的开
		头按n可以继续进行搜索，找出该字符下一次出现的位置。
			用？取代/，可以实现反向搜索。
			无论搜索方向如何，当达到文件尾或开头时，会继续循环搜索。
	(6)撤销和重复
		u：撤销前一条命令的结果
		.:重复执行上次执行的结果
	(7)文本选中
		v：字符选中命令
		V：行选中命令
5、文本输入模式中的编辑操作

		A：在光标所在行的末尾输入正文

6.末行模式
	:q		未做修改的情况下退出
	:q!		放弃所有修改，退出
	:wq		保存并退出
	:wq!	强制保存退出
	:n 		将光标移动到第n行

	在命令模式中，可以规定命令操作的行号范围。数值用来指定绝对行号；字符"."
	表示光标所在行行号；字符"$"表示正文最后一行的行号；
	
	:345w file 		将第345行写入文件file
	:3,5w file 		将第3行至第五行写入文件file
	:1,.w file 		将第一行至当前行写入文件file
	:w file 		将文件内容写入file文件，保持原有文件的内容不变
	:a,bw file 		将第a行至第b行写入文件file
	:r file 		读取file文件的内容
	:e file 		编辑新文件到file代替原有内容
	:f file 		将当前文件重新命名为file
	:f 				打印当前文件的名称和状态

(2)查找替换
	:s/str1/str2/		用str2替换行中首次出现的字符串str1
	:s/str1/str2/g		用str2替换行中所有出现的字符串str1
	:.,$ s/str1/str2/g	用str2替换当前行到末尾所有出现的字符串str1
	:1,$ s/str1/str2/g  用str2替换文件中所有出现的字符串str1
	:g/str1/s/str2/g 	用str2替换文件中所有出现的字符串str1
	
(3)删除文本中的内容

	:d 					删除光标所在行
	:3d					删除三行
	:.,$d 				删除当前行只正文的末尾
	:/str1/,/str2/d 	删除从字符串str1到str2的所有行
	
	
7.窗口分割

(1)启动vi时
	vi -o file file1 file2 ...
	vi -O file file1 file2 ...
	-o:水平分割窗口
	-O:垂直分割窗口
(2)在vi编辑屏幕时使用
	:vs 	垂直分割窗口
	:sp 	水平分割窗口

	Ctrl+w + 	扩大窗口
	Ctrl+w -	缩小窗口
	Ctrl+w h 	移动窗口到左边
	Ctrl+w j	移动窗口到下边
	Ctrl+w k    移动窗口到上边
	Ctrl+w l 	移动窗口到右边
	Ctrl+w Ctrl+w 	在窗口之间移动

(3)高级编辑指令
d	删除
y	复制
P	放置
c	修改
范围：
e	光标所在位置到改字的最后一个字母
w 	光标所在位置到下一个字的第一个字母 
b	光标所在位置到上一个字的第一个字母
S	光标所在位置到该行的最后一个字母
0 	光标所在位置到该行的第一个字母
)	光标所在位置到下一个句子的第一个字母
(	光标所在位置到该句子的第一个字母
}	光标所在位置到该段落的最后一个字母
{	光标所在位置到该段落的第一个字母
