1. 文件操作
（1）什么是文件？	 
		一组相关数据的集合，称之为文件

（2）文件的分类
	 	-:普通文件
	 	l:链接文件
	 	p:管道文件
	 	c:字符设备文件
	 	b:块设备文件
	 	d:目录文件
	 	
	 	可读/可写/可执行

（3）文件的属性
		r：可读
			当文件以可读属性打开时，该文件必须存在
			
	 	w：可写
	 		如果文件不存在，则以给定的文件名，创建文件
	 		若文件已存在，则清空原有内容，重新写入
	 		
	 	a：追加
	 		在文末写入内容
	 	
	 	r+/rb+/rt+:
	 	w+/wb+/wt+:
	 	a+/ab+/at+:
	 		
	 	二进制文件：（.dat）
	 		以数据在内存中的形式保存到文件
	 		所占空间较小，效率较高
	 		文件内容看不懂
	 		
	 	文本文件：（.txt）
	 		文本文件以 ASCII 码的形式保存数据
	 		占据空间较大，内容比较直观

2. 文件的相关操作
（1）文件结构
	 FILE 系统定义的结构体
	 文件内部指针
	 文件属性等
	 
	 FILE *fp;	//定义一个文件指针
	 对指针操作即对文件操作

（2）文件的打开和关闭
	a. 文件打开
	 fopen();
	 FILE *fopen(const char *path, const char *mode);
	 path：文件名（可加路径）
	 	   若带路径则是绝对路径或者以 .  .. 开头的路径
	 mode：r/w/a/r+/w+/a+
	 
	 返回值：
	 	成功：返回一个指向文件的文件指针	 
	 	失败：返回 NULL
	 
	b. 文件关闭
	 fclose();
	 int fclose(FILE *stream);

	 对于一个打开的文件，若操作结束后，必须关闭
	 否则文件内容会丢失


3. 文件的读写操作
（1）字符读写
	 fgetc()/fputc()
	 int fgetc(FILE *stream);
	 	从 stream 所对应的文件中，读取一个字符
	 int fputc(int c, FILE *stream);
	 	将 c 写入 stream 中
	 
	 练习：
	 	实现文件的拷贝：

（2）行读写
	 fgets()/fputs()
	 
（3）格式化读写
	 fscanf()/fprintf()

（4）块读写
	 fread()/fwrite()
