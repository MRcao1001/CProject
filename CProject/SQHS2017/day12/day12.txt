1. 函数
（1）定义
	 实现特定功能并且相对独立的某部分代码，将其抽象
	 出来作为一个独立的个体，在同一程序或其他程序中
	 可以多次使用的代码。
	 
	 功能独立的代码段

（2）函数的作用
     a. 可以多次重复利用
     b. 使代码更加简洁
     c. 使代码模块化
     d. 便于维护代码

（3）函数的分类
	 a. 定义角度
	 	系统函数
	 		由系统定义，用户直接使用即可，无需对其
	 		进行说明，但需要包含相应的头文件
	 		
	 		printf()/scannf()
	 		fgets()/fputs()
	 		strlen()/strcmp() ...
	 		
	 		math.h 中的几个函数
	 		sqrt()  --- 开方函数
	 		pow()   --- 次幂函数
	 	
	 	练习：
	 		分别求出: 2^0 ~ 2^10
	 		
	 		toupper()  转大写
	 		tolower()  转小写
	 			
	 	自定义函数
	 		用户根据需要，实现相关功能编写出来的函数
	 		使用之前需要做相应的说明
	 
	 b. 过程角度
	 	有返回值
	 		函数调用，执行函数体之后产生的结果
	 		将该值返回到主调函数。
	 		
	 		主调函数
	 		被调函数
	 		
	 	函数基本定义形式：
	 		类型说明符 函数名称(参数列表)
	 		{
	 			//函数体
	 			//返回值
	 		}
	 	
	 		int swap(int a, int b)
	 		{
	 		
	 			return a,b;	//OK,此处为逗号表达式
	 			//return (a,b);
	 		}
	 	
	 	无返回值
	 		无需向主调函数返回执行结果
	 		一般为 void 类型
	 		
	 		void test()
	 		{
	 			//函数体
	 			return;
	 		}
	 		
	 		void testOne()
	 		{
	 			//函数体
	 		}
	 	
	 c. 数据传送角度
	 	有参函数
	 	类型说明符 函数名称(参数列表)
	 		{
	 			//函数体
	 			//返回值
	 		}
	 		
	 	实际参数：
	 		常量表达式或者有确切值的表达式
	 		或着函数调用
	 		
	 	形式参数：
	 		就是变量定义
	 	
	 	发生函数调用时，由实参到形参的过程，就是
	 	赋值过程，即把实参的值赋给形参
	 	
	 	形参列表中，若有多个形参则之间用逗号隔开
	 	
	 	实参和形参在数量，顺序，类型要严格
	 	一一对应
	 	
	 	无参函数
		 	没有形参的函数
		 	
		 	int test()
		 	{
		 		printf("CHINA\n");
		 		return 0;
		 	}
